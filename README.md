StyleCart – Online Dress Shopping Website

Overview

StyleCart is a Django-based online dress-shopping web application developed as part of my web programming studies. The project provides a simple, user-friendly platform where users can browse dresses, view product details, add items to their cart, place orders, and manage their purchases.

Features

* User-friendly home page and product catalog
* Product details with image, description, available sizes, material, and price
* Shopping cart with add and remove functionality
* Buy Now option with size, quantity, and delivery address selection
* Order confirmation after successful purchase
* View all ordered items
* Cancel previously placed orders
* Customer support page for sending queries
* Responsive interface designed using HTML, CSS, and JavaScript

Technologies Used

* Backend: Python, Django
* Frontend: HTML5, CSS3, JavaScript
* Database: SQLite
* IDE: PyCharm
* Version Control: Git & GitHub

Installation

1. Clone the repository.
2. Navigate to the project directory.
3. Install the required packages:

pip install -r requirements.txt

1. Apply database migrations:

python manage.py migrate

1. Start the development server:

python manage.py runserver

1. Open your browser and visit:

http://127.0.0.1:8000/

Functionalities
